# Людина в контурі

«Людина в контурі» означає, що окремі рішення або дії робочого процесу потребують перевірки, внеску, схвалення чи виправлення людиною.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Участь людини слід розміщувати у значущих точках контролю, а не додавати як нечітку остаточну перевірку. Схвалення особливо цінне перед незворотними, фінансовими, юридичними, публічними, чутливими до безпеки або високовпливовими діями.

## Поширені шаблони

- Схвалення запропонованого плану перед виконанням.
- Перевірка згенерованого коду або змін документа перед публікацією.
- Розв’язання випадків класифікації з низькою впевненістю.
- Підтвердження одержувачів, сум або дозволів перед зовнішньою дією.
- Ескалація повторюваних збоїв або конфліктів політик.

## Рекомендації з проєктування

Показуйте перевіряльнику докази, запропоновану дію, очікуваний ефект і відому невизначеність. Зробіть явними шляхи схвалення, відхилення й зміни. Зберігайте рішення у стані робочого процесу й не дозволяйте моделі його обходити.

## Компроміси й обмеження

Людська перевірка знижує пропускну здатність і може перетворитися на формальне схвалення, якщо сповіщення надто часті або погано пояснені. Перевіряльникам також може бракувати контексту для виявлення тонких помилок.

## Типові помилки

- Запитувати схвалення після того, як дія вже відбулася.
- Показувати лише резюме моделі замість первинних доказів.
- Вважати мовчання або сплив часу очікування схваленням.
- Використовувати людей для компенсації відсутньої детермінованої перевірки.
