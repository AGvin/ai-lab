# Модерація вмісту

Модерація виявляє, позначає, обмежує або спрямовує вміст відповідно до політик безпеки, права, платформи чи спільноти.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Модерація може відбуватися до оброблення моделлю, після генерації або на обох етапах. Системи поєднують правила, класифікатори, судження моделей, скарги й людську перевірку. Політики мають явно визначати категорії та дії, бо «небезпечність» залежить від контексту.

## Практичне застосування

- Фільтрувати заборонений ввід користувача.
- Перевіряти згенерований публічний вміст.
- Захищати неповнолітніх або робочі середовища.
- Передавати непевні випадки людям-модераторам.
- Забезпечувати межі політик конкретного продукту.

## Компроміси й обмеження

Системи модерації дають хибнопозитивні й хибнонегативні результати. Мова, культура, сарказм, цитати й освітній контекст ускладнюють класифікацію. Надто широкі фільтри можуть пригнічувати законний вміст.

## Типові помилки

- Використовувати один поріг для всіх категорій.
- Приховувати рішення без можливості оскарження.
- Сприймати цитування шкідливого вмісту як його підтримку.
- Застосовувати текстову модерацію до дій інструментів без окремого контролю.
