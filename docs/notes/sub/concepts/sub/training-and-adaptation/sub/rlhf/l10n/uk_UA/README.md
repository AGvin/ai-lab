# RLHF

Reinforcement Learning from Human Feedback, або RLHF, використовує дані людських переваг, щоб навчити модель поведінки, яку оцінюють як бажанішу.

## Переклади

- [English](../../)
- Українська

## Типовий pipeline

1. Supervised fine-tuning створює модель, що виконує інструкції.
2. Люди порівнюють або ранжують кандидатні відповіді.
3. Reward model навчається прогнозувати ці переваги.
4. Reinforcement learning оновлює мовну модель, щоб збільшити прогнозовану винагороду.

## Основна ідея

RLHF оптимізує поведінку моделі через вивчений сигнал винагороди, а не лише через пряму імітацію цільових токенів. Його широко використовують для покращення корисності й безпекової поведінки моделей-assistants.

## Компроміси й обмеження

Reward models — недосконалі наближення. Політика може використовувати слабкості reward model, ставати надмірно обережною або вивчати бажаний стиль без покращення правдивості. RLHF експлуатаційно складний і потребує ретельного контролю стабільності навчання.

## Типові помилки

- Припущення, що людська перевага дорівнює фактичній правильності.
- Нечіткі критерії анотації.
- Відсутність моніторингу reward hacking і регресій.
- Сприйняття RLHF як необхідного для малих доменних адаптацій.
