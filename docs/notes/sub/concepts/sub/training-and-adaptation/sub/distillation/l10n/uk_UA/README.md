# Дистиляція

Дистиляція навчає меншу або простішу модель-учня відтворювати корисну поведінку потужнішої моделі-вчителя.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Учитель може надавати розподіли ймовірностей, згенеровані відповіді, приклади reasoning, мітки чи синтетичні задачі. Учень навчається за цими сигналами й може зберегти частину можливостей учителя за нижчої вартості inference.

## Практичне застосування

- Створювати менші моделі для локального або edge-розгортання.
- Переносити task-поведінку з дорогої моделі.
- Генерувати навчальні мітки, коли людська анотація обмежена.
- Спеціалізувати компактну модель для вузького workflow.

## Компроміси й обмеження

Учень також успадковує помилки й упередження вчителя. Дистиляція не може зберегти кожну можливість зі зменшенням місткості моделі. Згенеровані навчальні дані можуть порушувати умови провайдера чи ліцензії моделі, тому важливі походження й права використання.

## Типові помилки

- Оцінювання лише на згенерованих учителем прикладах.
- Сприйняття вчителя як джерела істини.
- Ігнорування contamination даних і ліцензування.
- Очікування, що значно менший учень відповідатиме кожній можливості вчителя.
