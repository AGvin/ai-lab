# Synthetic Data

Training or evaluation data generated by rules, simulations, or models.

## Core idea

Training or evaluation data generated by rules, simulations, or models. In practical AI work, the term is useful because it names a specific part of the system rather than treating the model as a single opaque component. Understanding where it appears in the workflow makes configuration choices and failure analysis more precise.

## How it works

- Synthetic data is generated by rules, simulations, models, or transformations rather than collected directly from natural observations.
- Generation pipelines can target rare cases, balance classes, or create instruction-response pairs.
- Filtering and validation are needed before synthetic examples enter training or evaluation.

The exact implementation varies by model family, provider, and runtime. The important distinction is the role the concept plays in the end-to-end system and which inputs, state, or resources it changes.

## Why it matters

Synthetic Data affects how an AI system should be selected, configured, tested, or operated. It can influence output quality, resource requirements, reliability, or the amount of control available to the surrounding application.

## Practical uses

- Expand limited datasets and cover edge cases.
- Generate test cases or bootstrap adaptation data.

## Example

A stronger model generates candidate coding tasks that are compiled and tested before being accepted into a training set.

## Trade-offs and limitations

- Synthetic errors and model biases can be amplified during training.
- A dataset may look diverse while remaining constrained by the generator’s knowledge.

Do not evaluate this concept in isolation. Test it together with the actual model, data, runtime, tools, and workload that will be used in production or local experiments.

## Practical checklist

- What problem is Synthetic Data expected to solve in this workflow?
- Which inputs, settings, or resources does it depend on?
- How will success and failure be measured?
- What changes when the model, runtime, dataset, or context size changes?
