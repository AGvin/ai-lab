# Supervised fine-tuning

Supervised Fine-Tuning, або SFT, навчає модель на прикладах, що поєднують input чи інструкцію з бажаним output.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Модель навчається надавати вищу ймовірність цільовим відповідям із dataset. Для chat-моделей приклади зазвичай містять ролі, інструкції, контекст і відповіді assistant, відформатовані за chat template моделі.

## Практичне застосування

- Перетворити базову модель на модель, що виконує інструкції.
- Навчити спеціалізованих форматів і термінології.
- Покращити поведінку на підготовлених доменних прикладах.
- Підготувати модель до оптимізації переваг.

## Вимоги до dataset

Якісні різноманітні приклади цінніші за великі обсяги повторюваного тексту. Додавайте реалістичні помилки, edge cases і бажану поведінку за невизначеності. Вилучайте секрети, випадкові персональні дані й неправильні синтетичні відповіді.

## Компроміси й обмеження

SFT навчає імітувати приклади, а не незалежно перевіряти факти. Він може надмірно підлаштуватися під стиль, запам’ятати приклади або знизити продуктивність поза навчальним розподілом. Модель також може вивчити небажані шаблони, приховані в нібито правильних відповідях.

## Типові помилки

- Навчання на необроблених документах замість пар «інструкція — відповідь» для instruction-задачі.
- Використання згенерованих відповідей без валідації.
- Змішування несумісних chat templates.
- Оцінювання на майже дубльованих навчальних даних.
