# Instruction tuning

Instruction tuning навчає модель на різноманітних прикладах «інструкція — відповідь», щоб вона краще виконувала задачі, сформульовані природною мовою.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Попередньо навчену мовну модель переважно оптимізовано для продовження тексту. Instruction tuning змінює цю поведінку в бік відповіді на запити, дотримання описів задач і узагальнення між багатьма форматами інструкцій. Зазвичай його реалізують через supervised fine-tuning, а після нього може виконуватися оптимізація переваг.

## Практичне застосування

- Створити поведінку універсального assistant із базової моделі.
- Покращити виконання задач zero-shot.
- Навчити форматів відповідей і шаблонів відмов.
- Адаптувати модель до поширених інструкцій домену.

## Компроміси й обмеження

Instruction tuning може послабити початкову поведінку продовження тексту й додати упередження або обмеження instruction dataset. Модель може вивчити поверхневі шаблони відповідей, не стаючи фактично надійнішою.

## Типові помилки

- Використання вузького набору інструкцій з очікуванням широкого узагальнення.
- Ототожнення корисного тону з правильністю.
- Відсутність складних або неоднозначних прикладів.
- Ігнорування tokenizer і chat template, яких очікує модель.
