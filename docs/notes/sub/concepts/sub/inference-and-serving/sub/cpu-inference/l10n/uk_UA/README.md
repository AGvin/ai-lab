# Інференс на CPU

Інференс на CPU виконує більшість обчислень моделі на ядрах процесора загального призначення й у системній пам’яті.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Інференс великих мовних моделей часто обмежує пропускна здатність пам’яті, а не лише арифметична продуктивність. CPU можуть досить ефективно запускати квантовані моделі для розроблення, сервісів із малим навантаженням, ембедингів або пристроїв без придатних GPU. На продуктивність сильно впливають підтримка векторних інструкцій і пропускна здатність каналів пам’яті.

## Практичне застосування

- Запускати локальні моделі на обладнанні без дискретного GPU.
- Обслуговувати навантаження з малою паралельністю й великою системною RAM.
- Виконувати малі моделі ембедингів, класифікації чи переранжування.
- Використовувати шари CPU разом із частковим вивантаженням на GPU.

## Міркування щодо проєктування

Узгоджуйте кількість потоків із фізичними ядрами й поведінкою середовища. Надмірна кількість потоків створює конкуренцію. Використовуйте швидку пам’ять, належне NUMA-розміщення в багатосокетних системах і оптимізовані для архітектури процесора ядра.

## Компроміси й обмеження

CPU зазвичай має нижчу пропускну здатність генерації, ніж потужний GPU, але системна RAM може вмістити більші моделі за нижчою ціною. Високе завантаження може впливати на інші процеси й підвищувати енергоспоживання.

## Типові помилки

- Припускати, що більше потоків завжди підвищує швидкість.
- Порівнювати короткочасну продуктивність ноутбука зі сталою роботою сервера.
- Ігнорувати пропускну здатність пам’яті й топологію NUMA.
- Використовувати не оптимізовану універсальну збірку без векторних розширень.
