# Перетворення тексту на зображення

Генерація text-to-image створює зображення, зумовлене переважно описом природною мовою.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Промпт кодується в подання, що спрямовує генеративну модель. Його частини впливають на об’єкт, композицію, середовище, стиль, колір, камеру й освітлення, але модель не тлумачить текст як детерміновану специфікацію сцени.

## Практичний промптинг

- Спочатку вказуйте головний об’єкт і дію.
- Явно описуйте композицію й точку огляду.
- Після основної сцени додавайте середовище, освітлення й техніку.
- Використовуйте негативні обмеження лише за підтримки й потреби.
- Створюйте кілька кандидатів і вдосконалюйте найсильніший.

## Компроміси й обмеження

Текстові промпти дають гнучкий високорівневий, але слабкий точний контроль геометрії, ідентичності, типографіки й кількості об’єктів. Тлумачення промпту різниться між сімействами й версіями моделей.

## Типові помилки

- Перелічувати суперечливі стилі й напрями камери.
- Очікувати точного написання згенерованого тексту.
- Одночасно змінювати багато елементів промпту.
- Припускати незмінне перенесення формулювань між моделями.
