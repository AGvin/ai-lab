# Генерація аудіо

Генерація аудіо створює музику, мовлення, звукові ефекти, атмосферу чи інші хвильові форми з тексту, референсного аудіо, символьного входу або навчених латентних подань.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Моделі можуть генерувати аудіо безпосередньо, прогнозувати стиснені аудіотокени або застосовувати дифузію в поданні хвильової форми чи спектрограми. Умови можуть керувати вмістом, тривалістю, стилем, інструментами, голосом або характеристиками середовища.

## Практичне застосування

- Звукові ефекти й атмосфера для прототипів.
- Пошук музичних ідей і фонові доріжки.
- Синтетичне мовлення й голоси персонажів.
- Відновлення або продовження аудіо.
- Доповнення наборів даних.

## Компроміси й обмеження

Складно підтримувати довгу структуру, точний ритм, зрозумілий текст пісень і сталу ідентичність. Згенерований матеріал може нагадувати стилі з навчальних даних або містити артефакти. Права на навчальні дані, ідентичність голосу й комерційне використання різняться.

## Типові помилки

- Припускати, що згенерована музика автоматично не має правових ризиків.
- Використовувати клоновані голоси без згоди.
- Очікувати збереження довгої музичної структури за одну генерацію.
- Ігнорувати гучність, кліпінг і мастеринг перед публікацією.
