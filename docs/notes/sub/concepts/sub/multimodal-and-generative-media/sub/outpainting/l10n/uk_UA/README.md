# Домальовування назовні

Outpainting розширює зображення за початкові межі, генеруючи новий навколишній вміст.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Початкове зображення дає візуальний контекст, а більше полотно містить порожні замасковані ділянки. Модель продовжує текстури, перспективу, освітлення й сцену в новій області. Перекриття оригінальної та згенерованої ділянок допомагає уникати видимих швів.

## Практичне застосування

- Змінювати співвідношення сторін.
- Розширювати фони для банерів або макетів.
- Показувати більше середовища навколо об’єкта.
- Перекадровувати композицію без обрізання.
- Створювати панорамні розширення.

## Компроміси й обмеження

Модель може вигадувати неузгоджену геометрію або повторювати об’єкти. Великі розширення мають менше початкового контексту й частіше дрейфують. Обличчя й центральні об’єкти біля межі можуть змінитися без ретельного захисту.

## Типові помилки

- Розширювати надто багато за один прохід.
- Не давати промпту для нової ділянки сцени.
- Використовувати жорстку межу маски без перекриття.
- Очікувати фактичної реконструкції прихованих частин реальних об’єктів.
