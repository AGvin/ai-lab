# Перетворення зображення на зображення

Генерація image-to-image перетворює наявне зображення, використовуючи його як умову для композиції, вмісту чи стилю.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Початкове зображення кодується й частково зашумлюється, а потім відтворюється під керуванням промпту або структури. Параметр сили видалення шуму зазвичай визначає прив’язку виходу до джерела: нижче значення зберігає більше деталей, вище дозволяє більші зміни.

## Практичне застосування

- Змінювати стиль композиції.
- Поліпшувати ескіз або грубий рендер.
- Змінювати освітлення, матеріали чи середовище.
- Створювати варіанти зі збереженням макета.
- Готувати зображення до локального домальовування.

## Компроміси й обмеження

Сильні перетворення можуть змінити ідентичність, геометрію, текст або дрібні деталі. Слабкі можуть зберегти небажані артефакти. Повторні проходи накопичують дрейф і знижують точність.

## Типові помилки

- Використовувати високу силу видалення шуму, коли потрібно зберегти ідентичність.
- Очікувати попіксельного збереження від генеративного редагування.
- Ігнорувати зміни співвідношення сторін і препроцесингу.
- Робити багато послідовних перетворень без повернення до найкращого джерела.
