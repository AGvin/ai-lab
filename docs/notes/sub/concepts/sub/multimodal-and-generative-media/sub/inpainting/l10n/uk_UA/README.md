# Домальовування всередині

Inpainting повторно генерує вибрані замасковані ділянки зображення, використовуючи оточення як контекст.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Маска визначає пікселі, які можна змінити. Модель заповнює ділянку за промптом, сусіднім вмістом і необов’язковими референсними контролями. Розмиття маски й відступ допомагають поєднати згенеровану ділянку з оточенням.

## Практичне застосування

- Вилучати або замінювати об’єкти.
- Виправляти руки, обличчя, текст чи дрібні артефакти.
- Відновлювати пошкоджені ділянки.
- Змінювати одяг, елементи фону чи матеріали.
- Робити локальну зміну без перегенерації всього зображення.

## Компроміси й обмеження

Надто мала маска може зберегти структуру артефакту, а надто велика — зайве змінити оточення. Для переконливого результату освітлення, перспектива, тіні й текстура мають відповідати оригіналу.

## Типові помилки

- Маскувати лише видиму помилку без її тіні чи віддзеркалення.
- Не залишати контекстного відступу навколо ділянки.
- Очікувати точного розміщення об’єкта лише з тексту.
- Повторно редагувати стиснене зображення й накопичувати артефакти.
