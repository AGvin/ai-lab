# Мультимодальний контекст

Мультимодальний контекст — це сукупна текстова, візуальна, аудіо-, відео-, документна й інструментальна інформація, доступна моделі під час запиту.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Кожна модальність перетворюється на токени чи навчені ознаки й займає частину обмеженого бюджету контексту або обчислень. Зображення високої роздільності, довге аудіо чи великий PDF можуть стискатися, вибірково оброблятися, ділитися або узагальнюватися до передавання моделі.

## Практичне застосування

- Поєднувати знімки екрана з технічними запитаннями.
- Аналізувати документи з текстом, таблицями й діаграмами.
- Використовувати транскрипти аудіо разом із метаданими мовців.
- Обґрунтовувати агента знімками браузера й результатами інструментів.

## Компроміси й обмеження

Додавання модальностей не гарантує їх правильного поєднання моделлю. Важливі деталі можуть губитися під час OCR, вибірки кадрів, зміни розміру чи транскрибування. Суперечливі докази між модальностями потребують явного оброблення.

## Типові помилки

- Припускати, що завантажені медіа обробляються з повною точністю.
- Надавати багато недоречних зображень і розсіювати увагу.
- Ігнорувати витрати контекстного вікна на медіа.
- Довіряти інструкціям усередині недовірених документів чи знімків.
