# Retrieval і знання

Поняття для під’єднання моделей до зовнішньої інформації, придатних до пошуку документів і контексту, підкріпленого свідченнями.

## Переклади

- [English](../../)
- Українська

Поняття згруповано за практичним пріоритетом. Пріоритет впливає на порядок читання, а не на тематичне розміщення.

## Основні

- [`rag/`](../../sub/rag/l10n/uk_UA/) — Retrieval-Augmented Generation поєднує зовнішній retrieval із генерацією моделі.
- [`embeddings/`](../../sub/embeddings/l10n/uk_UA/) — числові представлення, що розміщують семантично пов’язані елементи поруч у векторному просторі.
- [`chunking/`](../../sub/chunking/l10n/uk_UA/) — поділ вихідного матеріалу на придатні до retrieval одиниці зі збереженням достатнього контексту.
- [`semantic-search/`](../../sub/semantic-search/l10n/uk_UA/) — retrieval на основі значення, а не лише точних збігів слів.
- [`hybrid-search/`](../../sub/hybrid-search/l10n/uk_UA/) — retrieval, що поєднує сигнали семантичного й лексичного пошуку.
- [`grounding/`](../../sub/grounding/l10n/uk_UA/) — обмеження згенерованих тверджень наданими свідченнями, інструментами чи авторитетними джерелами.

## Корисні

- [`vector-search/`](../../sub/vector-search/l10n/uk_UA/) — retrieval найближчих сусідів у векторах embeddings.
- [`vector-databases/`](../../sub/vector-databases/l10n/uk_UA/) — системи зберігання й індексування, оптимізовані для векторів і пошуку подібності.
- [`keyword-search/`](../../sub/keyword-search/l10n/uk_UA/) — лексичний retrieval за термінами, присутніми у вихідному тексті.
- [`reranking/`](../../sub/reranking/l10n/uk_UA/) — модель релевантності другого етапу, що змінює порядок отриманих кандидатів.
- [`citations/`](../../sub/citations/l10n/uk_UA/) — посилання, що пов’язують згенеровані твердження з підтвердними джерелами.
- [`metadata-filtering/`](../../sub/metadata-filtering/l10n/uk_UA/) — обмеження retrieval за атрибутами джерела, дати, версії, tenant або політики доступу.

## Спеціалізовані

- [`bm25/`](../../sub/bm25/l10n/uk_UA/) — імовірнісний метод лексичного ранжування, широко вживаний у retrieval документів.
- [`graph-rag/`](../../sub/graph-rag/l10n/uk_UA/) — RAG, що використовує структури графа та зв’язки для формування контексту.
- [`knowledge-graphs/`](../../sub/knowledge-graphs/l10n/uk_UA/) — структуровані представлення сутностей та їхніх зв’язків.
