# Retrieval and Knowledge

Concepts for connecting models to external information, searchable documents, and evidence-backed context.

Concepts are grouped by practical priority. Priority affects reading order, not thematic placement.

## Переклади

- [English](../../)
- Українська — поточна

## Essential

- [`rag/`](../../sub/rag/l10n/uk_UA/) — Retrieval-Augmented Generation combines external retrieval with model generation.
- [`embeddings/`](../../sub/embeddings/l10n/uk_UA/) — Numerical representations that place semantically related items near each other in vector space.
- [`chunking/`](../../sub/chunking/l10n/uk_UA/) — Dividing source material into retrievable units while preserving enough context for use.
- [`semantic-search/`](../../sub/semantic-search/l10n/uk_UA/) — Retrieval based on meaning rather than only exact word matches.
- [`hybrid-search/`](../../sub/hybrid-search/l10n/uk_UA/) — Retrieval that combines semantic and lexical search signals.
- [`grounding/`](../../sub/grounding/l10n/uk_UA/) — Constraining generated claims to supplied evidence, tools, or authoritative sources.

## Useful

- [`vector-search/`](../../sub/vector-search/l10n/uk_UA/) — Nearest-neighbor retrieval over embedding vectors.
- [`vector-databases/`](../../sub/vector-databases/l10n/uk_UA/) — Storage and indexing systems optimized for vectors and similarity search.
- [`keyword-search/`](../../sub/keyword-search/l10n/uk_UA/) — Lexical retrieval based on terms that appear in the source text.
- [`reranking/`](../../sub/reranking/l10n/uk_UA/) — A second-stage relevance model that reorders retrieved candidates.
- [`citations/`](../../sub/citations/l10n/uk_UA/) — Links or references that connect generated statements to supporting sources.
- [`metadata-filtering/`](../../sub/metadata-filtering/l10n/uk_UA/) — Restricting retrieval by attributes such as source, date, version, tenant, or access policy.

## Specialized

- [`bm25/`](../../sub/bm25/l10n/uk_UA/) — A probabilistic lexical ranking method widely used in document retrieval.
- [`graph-rag/`](../../sub/graph-rag/l10n/uk_UA/) — Retrieval-Augmented Generation that uses graph structures and relationships to assemble context.
- [`knowledge-graphs/`](../../sub/knowledge-graphs/l10n/uk_UA/) — Structured representations of entities and their relationships.
