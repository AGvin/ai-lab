# Цитування

Цитування пов’язує згенеровані твердження з джерелами або свідченнями, використаними для їх підтвердження.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Корисна цитата відстежувана, конкретна й розташована біля твердження, яке підтверджує. Вона має визначати джерело та, коли можливо, сторінку, розділ, діапазон рядків, record ID або timestamp. Генерація цитат має ґрунтуватися на metadata отриманого джерела, а не на вигаданому тексті посилання.

## Практичне застосування

- Підкріплені свідченнями огляди досліджень.
- Відповіді за політиками, юридичною чи технічною документацією.
- Audit trails для даних, виділених із файлів або баз даних.
- Інтерфейси, що дають читачам змогу переглядати підтвердні уривки.

## Перевірки якості цитат

- **Правильність:** цитований уривок підтверджує твердження.
- **Повнота:** важливі фактичні твердження мають свідчення.
- **Конкретність:** посилання вказує на відповідне місце.
- **Походження:** ідентичність і версію джерела збережено.

## Компроміси й обмеження

Цитата може вести до низькоякісного або застарілого джерела. Надмірне цитування погіршує читабельність, а недостатнє ускладнює перевірку. Деякі згенеровані твердження поєднують кілька джерел і потребують більше одного посилання.

## Типові помилки

- Дозвіл моделі вигадувати URLs, назви чи номери сторінок.
- Цитування документа, який обговорює тему, але не підтверджує твердження.
- Втрата offsets цитат після preprocessing документа.
- Сприйняття цитат як заміни оцінюванню джерела.
