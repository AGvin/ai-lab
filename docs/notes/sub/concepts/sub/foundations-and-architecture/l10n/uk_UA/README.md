# Основи й архітектура

Ключові поняття, що пояснюють, чим є сучасні моделі ШІ та як співвідносяться їхні головні архітектурні сімейства.

Поняття згруповано за практичним пріоритетом. Пріоритет визначає порядок читання, а не тематичне розміщення.

## Переклади

- [English](../../)
- Українська

## Основні

- [`foundation-models/`](../../sub/foundation-models/l10n/uk_UA/) — великі моделі багаторазового використання, навчені на широких даних і адаптовані до багатьох подальших задач.
- [`large-language-models/`](../../sub/large-language-models/l10n/uk_UA/) — орієнтовані на мову фундаментальні моделі, які прогнозують і генерують послідовності токенів.
- [`multimodal-models/`](../../sub/multimodal-models/l10n/uk_UA/) — моделі, які обробляють або генерують кілька модальностей, зокрема текст, зображення, аудіо чи відео.

## Корисні

- [`transformers/`](../../sub/transformers/l10n/uk_UA/) — архітектури нейронних мереж, побудовані навколо уваги й паралельного оброблення послідовностей.
- [`attention/`](../../sub/attention/l10n/uk_UA/) — механізм, що зважує важливість елементів вхідних даних для операції моделі.
- [`mixture-of-experts/`](../../sub/mixture-of-experts/l10n/uk_UA/) — розріджені архітектури моделей, які спрямовують кожен вхід через вибрані експертні компоненти.

## Спеціалізовані

- [`artificial-intelligence/`](../../sub/artificial-intelligence/l10n/uk_UA/) — широка галузь систем, що виконують задачі сприйняття, міркування, генерації або ухвалення рішень.
- [`machine-learning/`](../../sub/machine-learning/l10n/uk_UA/) — методи, які навчаються закономірностей із даних замість опори лише на явно запрограмовані правила.
- [`deep-learning/`](../../sub/deep-learning/l10n/uk_UA/) — машинне навчання на основі нейронних мереж із багатьма шарами оброблення.
- [`neural-networks/`](../../sub/neural-networks/l10n/uk_UA/) — параметризовані обчислювальні структури зі з’єднаних шарів або блоків оброблення.
- [`self-attention/`](../../sub/self-attention/l10n/uk_UA/) — увага, обчислена між позиціями однієї послідовності.
- [`encoder-decoder/`](../../sub/encoder-decoder/l10n/uk_UA/) — структури моделей, що кодують вхідні дані, генерують вихідні або поєднують обидві ролі.
- [`dense-and-sparse-models/`](../../sub/dense-and-sparse-models/l10n/uk_UA/) — щільні моделі активують більшість параметрів для кожного входу, а розріджені — вибрані підмножини.
