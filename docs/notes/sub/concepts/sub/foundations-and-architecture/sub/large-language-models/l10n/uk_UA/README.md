# Великі мовні моделі

Велика мовна модель — це нейронна мережа, навчена моделювати й генерувати послідовності токенів у значному масштабі.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Більшість сучасних LLM навчають прогнозуванню наступного токена або спорідненим цілям. Це створює широкі можливості роботи з мовою, кодом, міркуванням і доповненням закономірностей. Далі навчання за інструкціями й оптимізація вподобань перетворюють базову модель на кориснішого асистента або модель для задач.

## Можливості

- Генерувати, узагальнювати, перетворювати й класифікувати текст.
- Створювати й аналізувати код.
- Використовувати інструменти через структуровані виклики.
- Працювати з отриманими документами.
- Дотримуватися інструкцій природною мовою.

## Практичне застосування

LLM слід розглядати як імовірнісний компонент. Застосунки оточують її промптами, інструментами, пошуком, станом, перевіркою й дозволами. Вибір моделі залежить від вимог до якості, розміру контексту, затримки, вартості, розгортання й безпеки.

## Компроміси й обмеження

LLM можуть галюцинувати, неправильно виконувати точні обчислення й відтворювати упередження навчальних даних. Їхні знання не є автоматично актуальними чи атрибутованими. Більші моделі часто мають ширші можливості, але збільшують вартість і використання ресурсів.

## Типові помилки

- Сприймати згенерований текст як вихід бази даних.
- Припускати, що модель знає актуальну приватну інформацію.
- Використовувати LLM для детермінованих задач без перевірки.
- Порівнювати моделі лише за кількістю параметрів.
