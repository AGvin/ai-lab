# Машинне навчання

Машинне навчання — це сукупність методів, які засвоюють закономірності або правила ухвалення рішень із даних замість опори лише на запрограмовану вручну логіку.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Алгоритм навчання коригує параметри моделі, щоб зменшити цільову функцію на навчальних даних. Отриману модель потім оцінюють на небачених даних, щоб визначити, наскільки добре вона узагальнює поза прикладами, на яких її навчали.

## Поширені категорії

- **Навчання з учителем:** навчається на позначених парах вхідних і вихідних даних.
- **Навчання без учителя:** виявляє структуру без явних цільових міток.
- **Самокероване навчання:** створює навчальні сигнали із самих даних.
- **Навчання з підкріпленням:** засвоює поведінку з винагород і взаємодії.

## Практичне застосування

Машинне навчання корисне, коли правила складно написати, але існують репрезентативні дані й вимірювані результати. Типові застосування охоплюють класифікацію, рекомендації, прогнозування, виявлення аномалій, розпізнавання мовлення й генеративні моделі.

## Компроміси й обмеження

Модель може добре відповідати історичним даним, але зазнати невдачі після зміни умов. Якість даних і міток, витік даних, дисбаланс класів та дизайн оцінювання часто важливіші за складність алгоритму.

## Типові помилки

- Оцінювати на навчальних прикладах.
- Вважати кореляцію доказом причинності.
- Розгортати без моніторингу дрейфу даних.
- Вибирати складну модель до створення простого базового рівня.
