# Software

Software documentation for AI-related model platforms, inference tools, workflow engines, agents, assistants, automation tools, and models.

## Child nodes

- [`inference/`](./sub/inference/) — tools for local or self-hosted model execution.
- [`workflow-engines/`](./sub/workflow-engines/) — tools that build or execute AI workflows.
- [`agents/`](./sub/agents/) — AI coding agents and development assistants.
- [`assistants/`](./sub/assistants/) — conversational AI assistants.
- [`automation/`](./sub/automation/) — automation tools that can support AI-adjacent workflows.
- [`model-platforms/`](./sub/model-platforms/) — platforms for model discovery, hosting, datasets, or related tooling.
- [`models/`](./sub/models/) — model families and individual models.

## Documentation model

Each product, model, platform, assistant, agent, or tool should have one canonical documentation node.

Use child nodes under that canonical node only when a specific usage scenario has enough real content to justify a separate page.
