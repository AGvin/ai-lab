# Code Editors

Локалізована точка входу для code editors та editor extension ecosystems, що використовуються для AI-assisted development.

## Переклади

- [English](../../)
- Українська

## Дочірні вузли

- [`cursor/`](../../sub/cursor/l10n/uk_UA/) — редактор коду з вбудованою інтеграцією ШІ.
- [`vs-code/`](../../sub/vs-code/l10n/uk_UA/) — редактор коду з екосистемою розширень для workflows за допомогою ШІ.

## Область

Використовуйте цей вузол для редакторів коду, редакторів із ШІ та спеціалізованих для редактора екосистем розширень.

Не розміщуйте редактор коду в `agents/` лише через наявність функцій ШІ. Використовуйте `agents/`, коли канонічний об’єкт є агентоподібною системою, а поведінка агента — його основною задокументованою роллю.
