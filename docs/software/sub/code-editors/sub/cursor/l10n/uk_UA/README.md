# Cursor

Локалізована точка входу для AI-enabled code editor.

## Переклади

- [English](../../)
- Українська

## Metadata

```text
Тип ресурсу: редактор коду з підтримкою ШІ
Основний сценарій: програмування за допомогою ШІ у спеціалізованому редакторі
Модель доступу: desktop-застосунок та/або hosted service
Експлуатаційна вимога: локальна робоча станція й доступ до облікового запису/сервісу
Режими інтеграції: desktop UI, editor workflow, інтеграції моделей/провайдерів
Джерело: https://cursor.com/
Примітки щодо ризиків: До роботи з чутливим кодом перевірте ціни, доступність вихідного коду, обробку даних, провайдерів моделей і поведінку доступу до репозиторію.
```

## Призначення

Використовуйте Cursor для програмування за допомогою ШІ, навігації кодовою базою, редагування й development workflows.

## Релевантність для ШІ

Cursor релевантний AI Lab як редактор коду з вбудованою функціональністю ШІ, а не як самостійний агент ШІ.

## Режими розгортання

- Локальна робоча станція
- Hosted SaaS або hybrid behavior: перевірте актуальну поведінку продукту

## Апаратне прискорення

- Зазвичай не стосується локального середовища

## Режими інтеграції

- Desktop UI
- Editor workflow
- Інтеграції моделей/провайдерів: перевірте актуальні варіанти
