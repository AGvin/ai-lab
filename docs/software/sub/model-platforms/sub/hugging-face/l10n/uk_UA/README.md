# Hugging Face

Локалізована точка входу для model, dataset, Spaces та AI tooling platform.

## Переклади

- [English](../../)
- Українська

## Metadata

```text
Тип ресурсу: платформа моделей
Модель доступу: хостингова платформа й локальне використання завантажених артефактів
Хостингові артефакти: моделі, datasets, Spaces і пов’язані інструменти
Провайдер: Hugging Face
Джерело: https://huggingface.co/
Примітки щодо ризиків: До впровадження перевірте ліцензії моделей і datasets, походження, ризики небезпечних моделей та exposure хостингового сервісу.
```

## Призначення

Використовуйте Hugging Face для пошуку, хостингу й поширення моделей, datasets, demos та інструментів ШІ.

## Релевантність для ШІ

Hugging Face релевантна AI Lab як велика платформа моделей і datasets та як джерело артефактів моделей для локальних або self-hosted workflows.

## Режими розгортання

- Hosted SaaS
- Локальне використання завантажених артефактів
- Hybrid workflows

## Апаратне прискорення

- Залежить від завантаженої моделі й runtime

## Режими інтеграції

- Web UI
- Репозиторій моделей
- Репозиторій datasets
- API або SDK: перевірте актуальні interfaces до використання
