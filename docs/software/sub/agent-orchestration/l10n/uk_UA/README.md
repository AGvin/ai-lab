# Agent Orchestration

Локалізована точка входу для systems, frameworks, runtimes та control planes для coordinating або running AI agents.

## Переклади

- [English](../../)
- Українська

## Область

Використовуйте цей вузол для інструментів, основна задокументована роль яких — оркестрація агентів, а не один самостійний агент.

Приклади:

- developer control centers для coding agents;
- multi-agent frameworks;
- stateful agent workflow runtimes;
- платформи, що координують agents, tools, backends, tasks або approval gates.

Використовуйте `agents/`, коли канонічний об’єкт є самостійним агентом або агентоподібною системою.

## Інструменти

- [`autogen/`](../../sub/autogen/l10n/uk_UA/) — multi-agent conversation framework для agents, tools, виконання коду й human-in-the-loop workflows.
- [`crewai/`](../../sub/crewai/l10n/uk_UA/) — framework і платформа multi-agent orchestration.
- [`langgraph/`](../../sub/langgraph/l10n/uk_UA/) — low-level orchestration framework і runtime для довготривалих stateful agents.
- [`openhands/`](../../sub/openhands/l10n/uk_UA/) — self-hosted developer control center для coding agents та automations.
