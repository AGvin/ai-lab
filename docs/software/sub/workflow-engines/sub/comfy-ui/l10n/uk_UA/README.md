# ComfyUI

Локалізована точка входу для node-based AI image generation workflow UI.

## Переклади

- [English](../../)
- Українська

## Metadata

```text
Тип ресурсу: workflow engine
Основний сценарій: вузлові AI workflows генерації зображень
Модель доступу: локальне або self-hosted ПЗ
Ліцензія: GPL-3.0
Модель вихідного коду: Open source
Експлуатаційна вимога: локальна робоча станція або self-hosted сервер
Режими інтеграції: web UI, workflow files, custom nodes
Апаратне прискорення: CPU; NVIDIA CUDA або інше прискорення залежить від установлення
Джерело: https://github.com/Comfy-Org/ComfyUI
Примітки щодо ризиків: До production перевірте джерело встановлення, походження extensions, довіру до custom nodes, ліцензування моделей, походження workflow files і network exposure.
```

## Призначення

Використовуйте ComfyUI для створення й запуску вузлових workflows генерації зображень.

## Релевантність для ШІ

ComfyUI релевантний AI Lab як практичне workflow environment для генерації зображень, експериментів і повторно використовуваних workflow templates.

## Режими розгортання

- Локальна робоча станція
- Self-hosted сервер

## Апаратне прискорення

- CPU
- NVIDIA CUDA: перевіряйте для конкретного встановлення
- Інше прискорення: перевіряйте для конкретного встановлення

## Режими інтеграції

- Web UI
- Workflow files
- Custom nodes

## Шаблони workflow

Спершу документуйте ComfyUI workflow templates як посилання на цій сторінці. Створюйте дочірній вузол документації лише за наявності достатнього реального вмісту для окремої сторінки.
