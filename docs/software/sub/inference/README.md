# Inference

Tools for local workstation and self-hosted model execution.

## Child nodes

- [`ollama/`](./sub/ollama/) — model runtime for local and self-hosted execution.
- [`lm-studio/`](./sub/lm-studio/) — desktop application and local server for running AI models.

## Scope

Use this node for software whose primary role in this repository is running or serving models.

A tool can also provide a UI, API, or integration surface, but it should still have one canonical page here when model execution is its main documented role.
