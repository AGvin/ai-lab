# Inference

Локалізована точка входу для інструментів локального або self-hosted model execution.

## Переклади

- [English](../../)
- Українська

## Дочірні вузли

- [`ollama/`](../../sub/ollama/l10n/uk_UA/) — runtime моделей для локального й self-hosted виконання.
- [`lm-studio/`](../../sub/lm-studio/l10n/uk_UA/) — desktop-застосунок і локальний сервер для запуску моделей ШІ.

## Область

Використовуйте цей вузол для ПЗ, основна роль якого в репозиторії — запуск або обслуговування моделей.

Інструмент також може надавати UI, API чи поверхню інтеграції, але повинен мати одну канонічну сторінку тут, якщо виконання моделей є його основною задокументованою роллю.
