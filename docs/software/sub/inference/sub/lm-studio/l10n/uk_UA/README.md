## Призначення

Використовуйте LM Studio для локального запуску моделей, експериментів із завантаженими моделями й надання локального API інструментам, що можуть підключатися до локальних model backends.

## Релевантність для ШІ

LM Studio релевантна AI Lab як середовище локального inference і backend для локальних development workflows за допомогою ШІ.
