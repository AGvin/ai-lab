# ChatGPT

ChatGPT — розмовний AI-продукт OpenAI для chat, research, роботи з files, multimodal tasks, connected applications, повторно використовуваних skills і tool-enabled workflows.

## Переклади

- [English](../../)
- Українська

## Роль продукту

Використовуйте цю сторінку для ChatGPT як продукту-асистента. Для repository-oriented coding agent і development workflow використовуйте сторінку [OpenAI Codex](../../../../../agents/sub/openai-codex/l10n/uk_UA/).

Можливості ChatGPT залежать від plan, workspace policy, region, вибраної model і product surface. Перед припущенням, що певний connector, plugin, skill або administrative control доступний, перевірте активні account і workspace settings.

## Plugins і підключені можливості

Plugins OpenAI — це встановлювані capability packages, які можуть містити skills, apps, app templates, connectors або MCP configuration. Точні компоненти й permissions відрізняються залежно від plugin.

Перед установленням перевірте publisher, source, bundled skills, connected accounts, OAuth scopes, MCP servers, network destinations та update policy.

## Примітки щодо оцінювання

Під час оцінювання workflow ChatGPT зафіксуйте:

- plan і тип workspace;
- вибрану model або mode;
- увімкнені skills і plugins;
- connected apps і надані scopes;
- доступ до files і network;
- approval behavior для consequential actions;
- чи доступна та сама configuration на desktop, web і mobile;
- дату перевірки.

Не припускайте, що skill надає доступ до tools або connected systems. Host, workspace policy, user authorization і активні connectors визначають, які operations доступні.
