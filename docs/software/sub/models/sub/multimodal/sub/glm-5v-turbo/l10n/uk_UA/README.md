## Попередження

Ця модель пов’язана із Z.ai / Zhipu AI з Китайської Народної Республіки. Залежно від сценарію використання до впровадження оцініть ризики supply chain, приватності, права, data governance й геополітики.

## Призначення

Використовуйте цю сторінку для відстеження GLM-5V-Turbo як кандидатної мультимодальної моделі для AI Lab.

## Релевантність для ШІ

GLM-5V-Turbo релевантна AI Lab, бо орієнтована на можливості мультимодальних агентів: сприйняття й reasoning щодо зображень, відео, вебсторінок, документів, GUIs, coding tasks, tool use та agent frameworks.
