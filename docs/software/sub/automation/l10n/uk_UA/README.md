# Automation

Локалізована точка входу для automation tools, що можуть підтримувати AI-adjacent workflows.

## Переклади

- [English](../../)
- Українська

## Дочірні вузли

- [`playwright/`](../../sub/playwright/l10n/uk_UA/) — framework автоматизації браузера й наскрізного тестування вебзастосунків.

## Область

Використовуйте цей вузол для ПЗ автоматизації, релевантного AI Lab через підтримку workflows за допомогою ШІ, валідації, керування браузером або суміжних сценаріїв автоматизації.

Не вважайте кожний універсальний інструмент автоматизації автоматично належним до цієї області. Інструмент повинен мати чітку причину, пов’язану чи суміжну зі ШІ, для документування тут.
