# Automation

Automation tools that can support AI-adjacent workflows.

## Child nodes

- [`playwright/`](./sub/playwright/) — browser automation and end-to-end web testing framework.

## Scope

Use this node for automation software that is relevant to AI Lab because it can support AI-assisted workflows, validation, browser control, or related automation scenarios.

Do not treat every general automation tool as in scope by default. The tool must have a clear AI-related or AI-adjacent reason to be documented here.
