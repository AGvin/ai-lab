## Overview

Lovable is a full-stack AI development platform for building web applications from natural language.

It generates editable application code and supports the lifecycle from prototype to deployment, with collaboration, GitHub sync, integrations, and governance features.

## Fit for AI Lab

Lovable belongs under `agents/` because its main documented behavior is agent-like software creation from natural-language instructions.

It is broader than a coding assistant and closer to an app-building agent platform.

Use Lovable as a reference point for:

- natural-language app generation;
- full-stack generated application workflows;
- GitHub-connected app-building platforms;
- low-code or vibe-coding agent products;
- comparison with Replit Agent and Bolt.

## Evaluation notes

Evaluate before adoption:

- generated code ownership and export model;
- GitHub sync and repository boundaries;
- workspace role and permission controls;
- database, authentication, and integration security;
- deployment and production-change review workflow;
- generated dependency and vulnerability review;
- suitability for prototypes versus maintained production systems.
