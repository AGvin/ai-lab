# Інструменти безпеки застосунків

Інструменти безпеки застосунків на основі ШІ для тестування, аналізу, пріоритизації та виправлення вразливостей програмного забезпечення.

## Переклади

- [English](../../)
- Українська

## Дочірні вузли

- [`keygraph/`](../../sub/keygraph/l10n/uk_UA/) — комерційна агентна AppSec-платформа для безперервного аналізу, white-box- і black-box-пентестингу, керування знахідками, виправлення та перевірки.

## Область

Використовуйте цей вузол для продуктів на основі ШІ, основна задокументована роль яких — безпека застосунків упродовж життєвого циклу розробки програмного забезпечення.

Самостійні автономні або напівавтономні агенти безпеки розміщуйте в [`agents/`](../../../../../agents/l10n/uk_UA/). Наприклад, Shannon задокументовано там, оскільки це самостійний агент пентестингу, тоді як Keygraph належить сюди як ширша AppSec-платформа, що використовує Shannon як один із компонентів.

Коли безкоштовна редакція або редакція з відкритим кодом надає лише вузьку частину значно ширшого комерційного продукту, явно документуйте функціональний розрив і враховуйте його як негативний чинник вибору, а не подавайте редакції як рівнозначні.
