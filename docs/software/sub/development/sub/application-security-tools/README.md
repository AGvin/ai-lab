## Scope

Use this node for AI-enabled products whose primary documented role is application security across the software-development lifecycle.

Keep standalone autonomous or semi-autonomous security agents under [`agents/`](../../../agents/). For example, Shannon is documented there because it is a standalone pentesting agent, while Keygraph belongs here because it is a broader AppSec platform that uses Shannon as one component.

When a free or open-source edition exposes only a narrow part of a substantially broader commercial product, document the feature gap explicitly and treat it as an adoption concern rather than presenting the editions as equivalent.
