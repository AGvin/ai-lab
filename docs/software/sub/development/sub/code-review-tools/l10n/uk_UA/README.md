# Інструменти перевірки коду

Неагентні інструменти для візуалізації, навігації, інспектування, анотування та перевірки diff і наборів змін коду в процесах розробки за допомогою ШІ.

## Переклади

- [English](../../)
- Українська

## Інструменти

- [`hunk/`](../../sub/hunk/l10n/uk_UA/) — review-first terminal diff viewer для наборів змін, створених агентами або людьми.

## Область

Використовуйте цей вузол, коли основна роль інструмента полягає у представленні або підтримці code review, а не в самостійній роботі як review agent.

Зберігайте агентоподібні review-системи, такі як CodeRabbit і Qodo, у [`agents/`](../../../../agents/l10n/uk_UA/). Інструменти життєвого циклу, що визначають специфікації, плани, завдання або quality gates, належать до [`development-workflows/`](../../../development-workflows/l10n/uk_UA/).
