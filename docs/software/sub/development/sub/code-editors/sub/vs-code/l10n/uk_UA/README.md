# Visual Studio Code

Локалізована точка входу для extensible code editor for AI-assisted workflows.

## Переклади

- [English](../../)
- Українська

## Metadata

```text
Тип ресурсу: редактор коду
Основний сценарій: розширюване середовище редагування коду
Модель доступу: desktop-застосунок
Експлуатаційна вимога: локальна робоча станція
Режими інтеграції: desktop UI, editor extensions, локальні або хостингові інтеграції
Джерело: https://code.visualstudio.com/
Примітки щодо ризиків: До роботи з чутливим кодом перевірте джерела й дозволи extensions, workspace trust, telemetry settings і доступ до model backend.
```

## Призначення

Використовуйте Visual Studio Code як редактор коду, який можна розширити пов’язаними зі ШІ extensions та інтеграціями локальних або хостингових моделей.

## Релевантність для ШІ

Visual Studio Code релевантний AI Lab, бо його екосистема extensions може підтримувати coding workflows за допомогою ШІ.

## Екосистема розширень

Документуйте спеціалізовані для редактора extensions у дочірньому вузлі `extensions/`.
