# AI Lab

Практична лабораторія ШІ для моделей, інструментів, локального інференсу, бенчмарків, нотаток про обладнання та робочих процесів автоматизації.

## Переклади

- [English](../../../)
- Українська

## Статус

Цей репозиторій є активною публічною лабораторією документації. Вміст додається поступово, коли тема має реальні нотатки, джерела або практичну цінність для оцінювання.

## Мапа документації

### Почати звідси

- [`overview/`](../../sub/overview/l10n/uk_UA/) — модель документації репозиторію, поточна структура, правила для ресурсів і настанови щодо розширення.
- [`disclaimer/`](../../sub/disclaimer/l10n/uk_UA/) — обмеження інформації репозиторію, суб’єктивних оцінок, гарантій, відповідальності та практичного використання.

### Програмне забезпечення

- [`software/`](../../sub/software/l10n/uk_UA/) — пов’язане зі ШІ програмне забезпечення, інструменти розробки, платформи моделей, засоби інференсу, агенти, автоматизація та виконувані робочі процеси.
  - [`development/`](../../sub/software/sub/development/l10n/uk_UA/) — програмне забезпечення для створення, структурування, інспектування та перевірки змін у розробці за допомогою ШІ.
    - [`code-editors/`](../../sub/software/sub/development/sub/code-editors/l10n/uk_UA/) — редактори коду та екосистеми розширень для розробки за допомогою ШІ.
    - [`development-workflows/`](../../sub/software/sub/development/sub/development-workflows/l10n/uk_UA/) — інструменти, що структурують постачання змін через специфікації, плани, задачі, валідацію та етапи реалізації.
    - [`code-review-tools/`](../../sub/software/sub/development/sub/code-review-tools/l10n/uk_UA/) — неагентні інструменти для інспектування, навігації, анотування та перевірки наборів змін коду.
  - [`inference/`](../../sub/software/sub/inference/l10n/uk_UA/) — інструменти для локального або самостійно розгорнутого виконання моделей.
  - [`workflow-engines/`](../../sub/software/sub/workflow-engines/l10n/uk_UA/) — інструменти для побудови або виконання робочих процесів ШІ.
  - [`agents/`](../../sub/software/sub/agents/l10n/uk_UA/) — агентоподібні системи ШІ, де агентна поведінка є основною задокументованою роллю.
  - [`agent-orchestration/`](../../sub/software/sub/agent-orchestration/l10n/uk_UA/) — системи, фреймворки, середовища виконання та площини керування для координації або запуску агентів ШІ.
  - [`assistants/`](../../sub/software/sub/assistants/l10n/uk_UA/) — розмовні асистенти ШІ.
  - [`automation/`](../../sub/software/sub/automation/l10n/uk_UA/) — інструменти автоматизації, що підтримують пов’язані зі ШІ робочі процеси.
  - [`model-platforms/`](../../sub/software/sub/model-platforms/l10n/uk_UA/) — платформи для пошуку й розміщення моделей, наборів даних і пов’язаних інструментів ШІ.
  - [`models/`](../../sub/software/sub/models/l10n/uk_UA/) — сімейства моделей та окремі артефакти моделей.

- [`benchmarks/`](../../sub/notes/sub/benchmarks/l10n/uk_UA/) — довідкові ресурси бенчмарків і рейтингів для оцінювання моделей, провайдерів і систем ШІ.
- [`comparisons/`](../../sub/notes/sub/comparisons/l10n/uk_UA/) — порівняння для підтримки рішень щодо моделей, інструментів, робочих процесів, платформ і систем ШІ.
