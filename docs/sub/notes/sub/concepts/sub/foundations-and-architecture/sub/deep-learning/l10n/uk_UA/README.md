# Глибоке навчання

Глибоке навчання — це машинне навчання на основі багатошарових нейронних мереж, які засвоюють ієрархічні подання з даних.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Ранні шари часто вивчають локальні або прості закономірності, а пізніші поєднують їх у більш абстрактні ознаки. У мовних моделях шари перетворюють подання токенів через увагу й операції прямого поширення. У моделях зору шари можуть вивчати краї, текстури, форми й семантичні об’єкти.

## Практичне застосування

Глибоке навчання ефективне для високовимірних неструктурованих даних: тексту, зображень, аудіо й відео. Воно лежить в основі сучасних мовних і дифузійних моделей, розпізнавання мовлення та багатьох систем сприйняття.

## Вимоги

- Великі або ретельно підготовлені набори даних.
- Значні обчислювальні ресурси й пам’ять для навчання.
- Методи оптимізації, здатні працювати з багатьма параметрами.
- Оцінювання, що виявляє перенавчання і зсув розподілу.

## Компроміси й обмеження

Глибокі мережі можуть досягати високої якості, але дорого навчаються, складні для інтерпретації й чутливі до якості даних. Вони можуть засвоювати скорочені шляхи або упередження з навчального розподілу.

## Типові помилки

- Припускати, що глибше завжди означає краще.
- Ігнорувати простішу модель, яка задовольняє вимоги.
- Вважати результат на бенчмарку доказом стійкості.
- Недооцінювати вартість даних та інфраструктури.
