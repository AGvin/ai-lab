# Штучний інтелект

Штучний інтелект — широка галузь створення систем, що виконують задачі, пов’язані зі сприйняттям, мовою, міркуванням, прогнозуванням, генерацією, плануванням або ухваленням рішень.

## Переклади

- [English](../../)
- Українська

## Основна ідея

ШІ — це узагальнювальний термін, а не один метод реалізації. Експертну систему на основі правил, класифікатор машинного навчання, велику мовну модель і систему керування роботом можна назвати ШІ, хоча вони працюють по-різному.

## Основні підходи

- Символьні системи на основі явних правил і подань.
- Машинне навчання, що засвоює закономірності з даних.
- Алгоритми пошуку й планування.
- Імовірнісні моделі для міркування за невизначеності.
- Генеративні моделі тексту, зображень, аудіо чи відео.
- Гібридні системи, що поєднують навчені й детерміновані компоненти.

## Практичне застосування

Використовуйте термін достатньо точно, щоб визначити фактичні можливості й механізм. У рішенні щодо продукту чи архітектури з’ясовуйте, які модель, дані, інструменти, правила й засоби людського контролю створюють поведінку, замість сприйняття «на базі ШІ» як технічного опису.

## Компроміси й обмеження

Системи ШІ можуть автоматизувати складну роботу, але успадковують обмеження даних, цілей, архітектури моделі, умов розгортання й людського проєктування. Успішність в одній задачі не означає загального інтелекту чи надійності в не пов’язаних задачах.

## Типові помилки

- Вважати ШІ синонімом великих мовних моделей.
- Припускати автономну поведінку через генеративний інтерфейс.
- Називати статистичний результат розумінням без визначення цього твердження.
- Ігнорувати простіші й безпечніші детерміновані альтернативи.
