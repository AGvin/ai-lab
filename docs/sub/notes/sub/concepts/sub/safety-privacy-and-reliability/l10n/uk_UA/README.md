# Безпека, приватність і надійність

Поняття для керування поведінкою моделей, захисту даних і зменшення операційних ризиків систем ШІ.

## Переклади

- [English](../../)
- Українська

Поняття згруповано за практичним пріоритетом. Пріоритет визначає порядок читання, а не тематичне розміщення.

## Основні

- [`prompt-injection/`](../../sub/prompt-injection/l10n/uk_UA/) — вхід, створений для перевизначення чи маніпулювання інструкціями системи ШІ.
- [`indirect-prompt-injection/`](../../sub/indirect-prompt-injection/l10n/uk_UA/) — шкідливі або суперечливі інструкції в зовнішньому вмісті.
- [`trust-boundaries/`](../../sub/trust-boundaries/l10n/uk_UA/) — явне розділення довірених інструкцій, систем, користувачів і недовірених даних.
- [`least-privilege/`](../../sub/least-privilege/l10n/uk_UA/) — надання системі лише дозволів для поточної задачі.
- [`sandboxing/`](../../sub/sandboxing/l10n/uk_UA/) — ізоляція виконання для обмеження наслідків збоїв або шкідливої поведінки.
- [`secret-handling/`](../../sub/secret-handling/l10n/uk_UA/) — захист облікових даних, токенів, ключів та інших чутливих даних.
- [`data-privacy/`](../../sub/data-privacy/l10n/uk_UA/) — контроль збирання, оброблення, зберігання й передавання особистих або конфіденційних даних.
- [`retrieval-poisoning/`](../../sub/retrieval-poisoning/l10n/uk_UA/) — маніпулювання індексованими знаннями для повернення оманливого чи шкідливого контексту.

## Корисні

- [`guardrails/`](../../sub/guardrails/l10n/uk_UA/) — засоби перевірки, обмеження й моніторингу входів, виходів і дій моделі.
- [`data-residency/`](../../sub/data-residency/l10n/uk_UA/) — географічне або юрисдикційне місце зберігання чи оброблення даних.
- [`provenance/`](../../sub/provenance/l10n/uk_UA/) — відомості про походження, власність, перетворення й зберігання даних або артефактів моделі.
- [`content-moderation/`](../../sub/content-moderation/l10n/uk_UA/) — виявлення чи обмеження вмісту за політиками безпеки, права або платформи.

## Спеціалізовані

- [`jailbreaking/`](../../sub/jailbreaking/l10n/uk_UA/) — спроби обійти налаштовані обмеження моделі.
- [`model-alignment/`](../../sub/model-alignment/l10n/uk_UA/) — спрямування поведінки моделі до визначених цілей, цінностей і політик.
- [`data-poisoning/`](../../sub/data-poisoning/l10n/uk_UA/) — псування навчальних даних для погіршення чи маніпулювання поведінкою моделі.
