# FlashAttention

FlashAttention — це сімейство реалізацій уваги, створених для зменшення обміну з пам’яттю й підвищення ефективності прискорювачів із точним обчисленням уваги в межах підтримуваної числової поведінки.

## Переклади

- [English](../../)
- Українська

## Основна ідея

Стандартна увага може матеріалізувати великі проміжні матриці у швидкій пам’яті. FlashAttention перебудовує обчислення на блоки, що вміщуються у швидшу пам’ять кристала, зменшуючи читання й записи. Це передусім оптимізація реалізації, а не інша архітектура моделі.

## Практичне застосування

- Прискорювати оброблення промптів для довгих послідовностей.
- Зменшувати тимчасове використання пам’яті під час уваги.
- Уможливлювати більші пакети або контексти на підтримуваних GPU.
- Підвищувати ефективність навчання й інференсу в сумісних середовищах.

## Компроміси й обмеження

Користь залежить від довжини послідовності, обладнання, типу даних, розмірів голів і підтримки ядер. Короткі запити чи непідтримувані архітектури можуть майже не прискоритися. Різні покоління FlashAttention і ядра постачальників мають різні вимоги сумісності.

## Типові помилки

- Вважати FlashAttention методом квантування моделі.
- Припускати, що ввімкнення прапорця гарантує використання оптимізованого ядра.
- Порівнювати продуктивність без перевірки довжини послідовності й розміру пакета.
- Очікувати усунення пам’яті KV-кешу.
