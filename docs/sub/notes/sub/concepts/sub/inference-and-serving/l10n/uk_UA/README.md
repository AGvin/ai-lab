# Інференс і обслуговування моделей

Поняття для локального запуску навчених моделей або надання їх як сервісів, а також розуміння використання ресурсів і продуктивності.

Поняття згруповано за практичним пріоритетом. Пріоритет визначає порядок читання, а не тематичне розміщення.

## Переклади

- [English](../../)
- Українська

## Основні

- [`inference/`](../../sub/inference/l10n/uk_UA/) — запуск навченої моделі для оброблення входу й створення виходу.
- [`quantization/`](../../sub/quantization/l10n/uk_UA/) — зменшення числової точності для зниження вимог моделі до пам’яті й обчислень.
- [`numerical-precision/`](../../sub/numerical-precision/l10n/uk_UA/) — числовий формат ваг, активацій і обчислень моделі.
- [`model-formats/`](../../sub/model-formats/l10n/uk_UA/) — файлові формати й формати серіалізації для зберігання та завантаження артефактів моделі.
- [`gpu-offloading/`](../../sub/gpu-offloading/l10n/uk_UA/) — перенесення вибраних обчислень або шарів моделі на GPU, коли інша робота залишається деінде.
- [`kv-cache/`](../../sub/kv-cache/l10n/uk_UA/) — кешовані ключі й значення уваги, повторно використовувані під час авторегресійної генерації.
- [`latency/`](../../sub/latency/l10n/uk_UA/) — час, потрібний для створення відповіді або досягнення визначеного етапу виходу.
- [`throughput/`](../../sub/throughput/l10n/uk_UA/) — обсяг роботи моделі, виконаний за одиницю часу.
- [`performance-metrics/`](../../sub/performance-metrics/l10n/uk_UA/) — показники часу до першого токена, токенів за секунду, затримки й використання пам’яті.

## Корисні

- [`model-serving/`](../../sub/model-serving/l10n/uk_UA/) — надання інференсу через керований процес, API, чергу або сервіс середовища виконання.
- [`model-loading/`](../../sub/model-loading/l10n/uk_UA/) — переміщення артефактів моделі до RAM, VRAM або керованої середовищем пам’яті для виконання.
- [`cpu-inference/`](../../sub/cpu-inference/l10n/uk_UA/) — виконання обчислень моделі переважно на процесорах загального призначення.
- [`gpu-inference/`](../../sub/gpu-inference/l10n/uk_UA/) — виконання обчислень моделі переважно на графічних процесорах для паралельних навантажень.
- [`context-caching/`](../../sub/context-caching/l10n/uk_UA/) — повторне використання раніше обробленого контексту промпту для зменшення повторних обчислень.

## Спеціалізовані

- [`flash-attention/`](../../sub/flash-attention/l10n/uk_UA/) — реалізації уваги, оптимізовані для зменшення обміну з пам’яттю й підвищення ефективності GPU.
- [`continuous-batching/`](../../sub/continuous-batching/l10n/uk_UA/) — динамічне об’єднання активних запитів інференсу для ефективнішого обслуговування.
- [`speculative-decoding/`](../../sub/speculative-decoding/l10n/uk_UA/) — використання швидшого чернеткового процесу для пропонування токенів, які перевіряє цільова модель.
