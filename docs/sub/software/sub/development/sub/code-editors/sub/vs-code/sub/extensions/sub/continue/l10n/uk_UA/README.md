# Continue

Continue — розширення-помічник для програмування за допомогою ШІ в editor workflows.

## Переклади

- [English](../../)
- Українська

## Metadata

```text
Тип ресурсу: VS Code extension
Основний сценарій: програмування за допомогою ШІ в editor workflows
Модель доступу: editor extension із локальним або hosted model backend
Ліцензія: Apache-2.0
Модель вихідного коду: Open source
Експлуатаційна вимога: Visual Studio Code і model backend
Режими інтеграції: editor extension, local model backend, hosted model backend
Апаратне прискорення: Залежить від вибраного model backend
Джерело: https://www.continue.dev/
Примітки щодо ризиків: До роботи з непублічним кодом перевірте джерело extension, model backend, scope доступу до коду й telemetry або обробку даних.
```

## Призначення

Використовуйте Continue, щоб під’єднати workflows Visual Studio Code до model backends для chat, autocomplete і допомоги з програмуванням.

## Релевантність для ШІ

Continue релевантний AI Lab як editor extension, здатний під’єднувати VS Code до локальних або хостингових model backends.

## Режими розгортання

- Локальна робоча станція
- Hybrid: перевірте model backend і конфігурацію

## Апаратне прискорення

- Залежить від вибраного model backend

## Режими інтеграції

- VS Code extension
- Local model backend
- Hosted model backend

## Пов’язані інструменти

- LM Studio
- Ollama

## Пов’язані сценарії

- Локальне autocomplete з LM Studio.
- Використання локальних моделей з editor workflows.
