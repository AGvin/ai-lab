## Попередження

Ця модель пов’язана з організацією з Китайської Народної Республіки. Залежно від сценарію використання до впровадження оцініть ризики supply chain, приватності, права, data governance й геополітики.

## Призначення

Використовуйте цю сторінку для відстеження Z-Image як кандидатної моделі генерації зображень для AI Lab.
