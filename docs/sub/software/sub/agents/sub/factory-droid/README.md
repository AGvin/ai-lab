## Overview

Factory Droid is part of Factory's agentic software-development platform.

It is positioned around team engineering workflows rather than a standalone code-completion feature. Treat it as a software-engineering agent product that can participate in development, review, and automation workflows across repositories and team tools.

## Evaluation notes

Evaluate before adoption:

- account, team, and enterprise access model;
- repository and organization permission scopes;
- review, commit, pull-request, and automation boundaries;
- integrations with existing engineering systems;
- generated change auditability;
- credential and secret exposure;
- human approval gates for merge, deploy, and release workflows.
