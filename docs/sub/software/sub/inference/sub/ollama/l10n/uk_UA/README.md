Локалізована точка входу для local та self-hosted model runtime.

- [English](../../)
- Українська

## Metadata

```text

Модель доступу: локальне ПЗ
Ліцензія: MIT
Модель вихідного коду: Open source
Експлуатаційна вимога: локальна робоча станція або self-hosted сервер
Режими інтеграції: CLI, HTTP API, локальний сервіс

Примітки щодо ризиків: До production перевірте джерела й ліцензії моделей, доступність локального сервісу, network binding і дозволи розгортання.
```

## Призначення

Використовуйте Ollama для запуску підтримуваних моделей на локальній робочій станції чи сервері та надання їх локальним інструментам або workflows.

## Релевантність для ШІ

Ollama релевантна AI Lab як варіант локального й self-hosted inference для експериментів, development workflows і приватного використання моделей.

## Режими розгортання

- Локальна робоча станція
- Self-hosted сервер

- CPU
